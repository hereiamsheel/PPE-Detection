# PPE Vision - AI Powered Personal Protective Equipment Detection System

PPE Vision is an AI-powered workplace safety monitoring system that automatically detects Personal Protective Equipment (PPE) compliance using computer vision. The application allows users to upload images/videos or monitor live webcam feeds to identify whether workers are wearing required safety gear such as helmets, masks, gloves, and safety vests.

## Features

- Real-time PPE detection using AI
- Image and video upload processing
- Live webcam monitoring
- Compliance and violation analysis
- Interactive analytics dashboard
- Visual reports with charts and statistics
- Processed image/video output generation

## Technology Stack

- Python
- Flask
- OpenCV
- YOLO Object Detection
- HTML, CSS, JavaScript
- Tailwind CSS
- Chart.js

## System Workflow

1. Upload image/video or start live camera.
2. AI detection engine processes frames.
3. PPE compliance and violations are identified.
4. Results and processed outputs are generated.
5. Analytics dashboard displays safety insights.
